CCC-Reset

completely customizable css reset

Css resets have served the industry well for half a decade now. Injecting a little sanity into starting a css project.

But css resets have a problem. Because css resets wipe the slate clean a developer often needs to add a lot of the same type of css to every project.

CCC uses the power of SASS & Compass to give developers a customizable starting point that can be tailored to any project.

Three Reasons to love CCC

  • Easy

    Simply drop one file into your sass folder, import it into your sass stylesheet and your ready to go.

  • Quick

    CCC has a slew of sensible defaults so you can get stated with any project quickly without any setup.

  • Customizable

    By adding just a few variables to the top of your sass stylesheet you can easily customize your reset to start you off on any project.

How to

To start using CCC you will first need to have SASS & Compass installed. An install guide for sass can be found here and install instructions for compass can be found here.

Next you need to get your hands on a copy of the infamous _CCC-R.scss file and place it in your sass directory. You can download that file right here.

file download

When you have _CCC-R.scss safely in your sass folder you need to import it into your main stylesheet. This is easy to do, simply add this line to the top of your sass stylesheet.

@import "CCC-R";

Thats it your ready to go. When your stylesheet complies you'll have a lovely reset that gives you a few sensible typographic niceties.
