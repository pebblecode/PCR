The CCC Reset

Completely, Customizable, CSS Reset

A new kind of CSS reset

There are a lot of great CSS resets out there. Some reset everything to give you a blank slate, some normalize various different browsers, some give you a few useful starting points.

The basic problem with any of these approaches is that is you reset everything you end up writing a lot of the same css every time your start a project. With a blank slate type reset you end up putting back in sensible defaults
