PCR

PCR "pebble custom reset"

PCR saves time in your css kitchen by cooking up an à la carte reset for your project.

Based on Eric Meyers awesome css reset With a few extra entrées added by the the fine folks at pebble{code}

Want to see any other css delicacies added to the menu? Help us out and add an issue to the github repro here.

Dependencies

PCR requires SASS. If SASS is not already installed there are installation instructions here

Demo

To get the demo working locally.

  1. Clone the repo.
    $ git clone git@github.com:pebblecode/PCR.git
  2. Run $ sass --watch sass/ to tell sass to auto compile your css as you make changes.
  3. Edit variables at the top of /sass/style.scss (See bellow for a list of variables.)
  4. Open index.html. Refresh to see changes to your reset as they are complied.

Standard usage

To use PCR in your own project.

  1. Copy _PCR.scss into your projects sass directory.
  2. Add the line @import "PCR"; to your main .scss file.
  3. PCR is now ready to go. Without specifying any variables PCR will compile Eric Meyer's css reset with no additions.
  4. Add variables. To enable the PCR additions you need to add some variables to your .scss file, above @import "PCR";

Variables

The following variables can be added to your .scss file above
@import "PCR";, to give your reset additional rules.

$remove-list-styles

Set to 'true' to add the following code to your reset:

/* 'true' */
ul, ol {
    list-style: none;
}

$link-underline

Set to 'none' or 'hover' to add the following code to your reset:

/* 'none' */
a:link{
  text-decoration: none;
}

/* 'hover' */
a:link{
  text-decoration: none;
}
a:hover{
  text-decoration: underline;
}

$ins-color

Set a color to add the following code to your reset:

/* 'yellow' */
ins{
  background-color: yellow;
}

$focus-outline

Set an outline property to add the following code to your reset:

/* '1px dotted grey' */
:focus {
  outline: 1px dotted grey;
}

$abbr-dfn-styles

Set to 'true' to add the following code to your reset:

/* 'true' */
abbr[title], dfn[title] {
  border-bottom: 1px dotted;
  cursor: help;
}

$hr-styles

Set to 'true' to add the following code to your reset:

/* 'true' */
hr {
display: block;
height: 1px;
border: 0;
border-top: 1px solid #cccccc;
margin: 1em 0;
padding: 0;
}
